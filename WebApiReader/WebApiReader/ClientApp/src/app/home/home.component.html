Greetings,

This site returns the most recent articles posted to Hacker News.

The application consists primarily of an angular page, hacker-news.ts, that calls a web controller, NewsConroller. The controller in turn calls a middle-tier layer, WepApiReaderMT. The site was developed with code reuse in mind, so additional API sources should be fairly easy to add.
